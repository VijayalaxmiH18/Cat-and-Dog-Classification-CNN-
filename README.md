# Cat-and-Dog-Classification-CNN-

#### Kaggle Dataset Link :
https://www.kaggle.com/tongpython/cat-and-dog
(This data set has 8005 training images and 2023 testing images.)

### Dependencies:
* Google Colaboratory
* Keras 
* Tensorflow - 2.2.0
* Python - 3.6
* Pandas 
* Seaborn 
* Matplotlib
* Numpy 
* Scikit-learn
